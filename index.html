Random Seed Generator

How to use

This program will generate a random number by utilizing multiple active sources of entropy from a javascript client along with a digital signature from the user. These values will then be hashed together using a merkle tree to generate a random seed.

Requirement: User must enable microphone.

Data Length: (10 < n < 50) (how many points of data required to form a seed)

Stream Mode can be used to generate a stream of random numbers from the sources of entropy. This stream will take in data from all sources and create a merkle tree hash from each data set. Each merkle tree hash will be appended to the previous creating a randomly generated stream of data.
